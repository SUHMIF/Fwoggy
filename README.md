# Fwoggy

### Controls
X to Start, and Arrow keys to Move (to avoid birds, and eat insects to win).

### About
A nice and simple, back to basics bite-sized game perfect for work break boredom that I developed within a few hours whilst taking part in Unity Learn's Junior Programmer Pathway course as I was influenced by trending PICO-8 games!

Similar to my previous projects I designed the game, developed the sprites using Aseprite, and programmed the gameplay within Unity.

Thank you for playing!
